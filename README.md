## 1. 시나리오 제목
* 테니스 클럽 관리 시스템
<br>


## 2. 시나리오 (5~10줄)
* 테니스 클럽의 운영, 관리를 보조하는 시스템입니다.
* 클럽 구성원의 정보를 등록합니다. 처음 등록시에는 이름, 성별, 구력별 기본 레이팅 등 기본적인 정보를 입력하며 이러한 기본 정보는 클럽 내 대회 운영에 사용합니다. 사전에 설정된 레이팅 구간별로 그룹을 생성하여 대회를 진행하고, 이때 프로그램을 이용하여 사람이 직접 작성하는 것보다 간편하게 대진표를 생성할 수 있습니다. 또한 경기 결과에 따라 레이팅이 조정되도록 합니다. 시간이 지나 데이터가 누적될수록 실력이 비슷한 구성원들끼리 경쟁할 수 있고 실력차에 따른 대진 불균형에 대한 구성원들의 불만을 해소하는데 도움이 될 것입니다. 구성원의 정보는 프로그램을 통해 언제든 쉽게 접근이 가능하며 클럽 운영에 적극적으로 이용할 수 있도록 사전에 입력한 레이팅 등 뿐만 아니라 개인 승률, 상대 전적등도 조회 가능하도록 할 것입니다.

<br>


## 3. 예상 기능 및 메뉴 (최소 5개)
1. 구성원 데이터 관리
2. 구성원 데이터 조회
3. 레이팅 부여 및 계산
4. 레이팅 별 그룹 분류
5. 그룹별 대회 대진표 생성
6. 승률 및 상대 전적 기록 및 보관

## 🟩 [2차 과제: V1.0] 입출력 + 리스트 + 조건문
### **✨2차 과제 내용:**
  * 이번 과제는 클럽 구성원의 기본적인 정보를 입력하고 입력한 정보를 저장, 삭제하는 기능을 구현하였습니다. 또한 반복문과 다중if를 결합한 데이터 필터링을 통해 점수별 등급을 부여는 기능을 구현하였습니다.
  * input을 사용하여 회원의 이름, 성별, 점수를 입력
  * if-elif-else를 사용하여 점수별 등급 부여(데이터 필터링)
  * for 반복문을 사용하여 회원 정보 리스트에 저장 및 삭제
  * sum(), max() 등의 함수를 사용하여 회원 점수 통계 출력
  * for 반복문을 사용하여 등록한 회원 정보 출력

    
### **🤖 AI 파트너십 과정**
1. **내용 1: 코드 작성 후 리스트 출력 도움 요청**
    * **프롬프트 요약:** "(작성한 코드) 리스트를 세로로 출력하는 방법을 알려줘"
    * **적용 내용:** for반복문을 사용하여 리스트를 출력하여 세로로 보기좋게 출력하였음.

2. **내용 2: 누락된 과제 조건 탐색 및 코드 개선 질문**
    * **프롬프트 요약:** "누락시킨 과제 조건이 있는지 찾아주고 그 외에도 개선할 점이 있으면 알려줘"
    * **적용 내용:** 리스트 조작과 연산자 활용등 누락된 조건을 찾아 코드를 추가 작성하였음. 제미나이가 sort()의 잘못된 사용을 추천하여 그것을 제외하고 수용하여 코드를 수정함.

3. **내용 3: 의도하지 않은 복합대입연산 결과 관련 질문**
    * **프롬프트 요약:** "count_new=5일 때 count_members = 5가 아닌 10으로 나오는 이유를 알려줘"
    * **적용 내용:** "잘못 작성한 복합대입연산을 파악하고 수정함"

4. **내용 4: 삭제 기능 추가 관련 자문**
    * **프롬프트 요약:** "이름을 입력해서 회원 정보를 삭제할 수 있을까?"
    * **적용 내용:** "for반복문과 if구문을 활용하여 삭제 기능을 구현함"

5. **내용 5: indexError: list index out of range**
    * **프롬프트 요약:** "indexError: list index out of range 에러가 나오는 이유와 해결방안을 질문"
    * **적용 내용:** "리스트 인덱스 작동 방식을 학습하고 break를 사용하여 해결함"
    
    
### **🛠️ Troubleshooting & 기술 회고:**
(예시) 
1. **문제 1: 등록한 회원 수의 계산 결과가 예상대로 나오지 않은 경우**
    * **원인:** 회원 수가 아닌 인덱스 번호의 합을 구하는 식을 작성하였음.
    * **해결:** 복합대입연산자 += 뒤에 (i)가 아닌 (1)을 써서 해결함.
2. **문제 2: 삭제 기능에 활용한 반복문 오류 (indexError)**
    * **원인:** pop()을 활용하여 리스트 전체의 길이는 줄어들었는데 range의 작동방식이 삭제된 리스트의 길이를 반영하지 않아 에러가 발생함.
    * **해결:** break를 활용하여 회원을 삭제하고 반복문에서 탈출함.
